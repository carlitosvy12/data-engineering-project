## Goal
The goal of this project is to build a complete data engineering pipeline to analyze housing affordability in Spain. The pipeline processes salary and housing price data, cleans and integrates the datasets, generates affordability indicators, and makes the results available for analysis and visualization.

## Requirements

- Python 3
- pip
- Virtual environment (recommended)
